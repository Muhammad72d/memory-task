# Agentic AI with Long-Term Memory

This project implements an **agentic system** that integrates a local LLM (Ollama) with a long-term semantic memory using **Chroma** vector database. It is designed to remember facts, events, and perform semantic search for an intelligent agent.

---

## Features

- **Base Memory Class**: Shared memory engine for all memory types.
- **Semantic Memory**: Stores factual information with metadata.
- **Episodic Memory**: Stores events and interactions.
- **Working Memory**: Stores short-term conversation context.
- **Local LLM (Ollama)**: Provides reasoning and natural language understanding.
- **Vector Search**: Uses embeddings for semantic search in memory.
